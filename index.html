Installation

Get up and running with Central Search in minutes

# Clone the repository git clone https://github.com/sheikh-vegeta/SheikhBot.git cd SheikhBot # Install dependencies pip install -r requirements.txt

Alternatively, you can install using pip:

pip install central-search

Quick Start Guide

Start analyzing your website and improving SEO in just a few steps

1

Analyze a single site

Run a basic analysis of your website to get started.

python main.py crawl https://example.com
2

Configure SEO analysis

Customize the crawler behavior by editing config.yml to match your needs.

3

Generate reports

Create beautiful reports to view your SEO analysis results.

python main.py ghpages
4

Set up IndexNow integration

Configure IndexNow to instantly notify search engines about content changes.

python main.py indexnow genkey

Automated Monitoring with GitHub Actions

Set up regular SEO monitoring for your websites

The included GitHub Actions workflow allows you to automatically run Central Search on a schedule or manually:

  1. Fork the repository to your GitHub account
  2. Enable GitHub Actions in your repository settings
  3. Customize the workflow in .github/workflows/crawl.yml
  4. Push changes to your repository
  5. Access your SEO reports on GitHub Pages

You can manually trigger a crawl and specify URLs, crawl depth, and which crawlers to use directly from the GitHub Actions tab.
